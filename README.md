# Bella & Brava Website

Redesigned website for Bella & Brava Restaurant, Coolaney, Co. Sligo.

## File Structure

```
bella-brava-website/
├── index.html          ← Main HTML file (open this in a browser)
├── css/
│   └── style.css       ← All styles (colours, layout, animations)
├── js/
│   └── main.js         ← Scroll animations, sticky nav, active links
└── README.md
```

## How to Use

Simply open `index.html` in any modern web browser — no build tools or server required.

For live hosting, upload all files to your web host maintaining the same folder structure.

## Design

- **Colours:** Forest Green (`#1c3a2a`) + Warm Cream (`#f5ede0`) + Gold (`#c8a86b`)
- **Fonts:** Playfair Display (headings) · Cormorant Garamond (body) · Josefin Sans (labels/nav)
- **Images:** Sourced from Unsplash — replace with real restaurant photography for best results
